🌶️ Chatpata Junction
Chatpata Junction is a full-stack e-commerce web application designed for selling a wide range of pickles online. The project is developed using Django for backend functionality and MySQL as the database, along with HTML, CSS, and JavaScript for building a responsive and user-friendly interface.
The platform allows users to explore different categories of pickles such as vegetables, fruits, non-veg, and regional varieties. It provides a seamless shopping experience where users can search products, add them to the cart, proceed to checkout, and place orders easily.
The system also includes a secure authentication mechanism, enabling users to register, log in, and manage their accounts. Additionally, users can track their orders, while the admin panel provides complete control over product management, user handling, order updates, and feedback moderation.

🚀 Features
User Registration & Login System
Product Category Browsing
Search Functionality
Add to Cart & Cart Management
Checkout & Order Placement
Order Tracking System
Admin Dashboard (Products, Users, Orders, Feedback)
Feedback & Rating System
Responsive UI Design

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Django (Python)
Database: MySQL
Version Control: Git & GitHub
